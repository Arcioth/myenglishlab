1. Data Collection & Usage Policy

In compliance with the Chrome Web Store User Data Policy, we are transparent about our data practices. The "Pearson Solver" extension adheres to the principle of Data Minimization.

What We Do NOT Collect

  • We do NOT collect, store, or transmit your browsing history or visited URLs.
  • We do NOT collect or store your Pearson/MyEnglishLab credentials (username/password).
  • We do NOT store logs of the questions you solve on our servers.
  • We do NOT sell any data to third parties.

What We Do Collect

  • Extension Credentials: We process the specific username/password you created for this extension (via myenglishlab.blog) solely for authentication and credit management.
  • Ephemeral Page Content: When you click "Solve", the text from the current exercise is temporarily processed to generate an answer. This data is not retained after the solution is provided.

2. Third-Party Data Processors

To provide AI-powered solutions, this extension transmits data to the following service provider:

  • Google Gemini API (Generative AI): The text content of English exercises is sent securely to Google's API for analysis.
    • Purpose: To understand the context and generate the correct answer key.
    • Privacy: Data sent to the API is subject to Google's Privacy Policy.

3. Permissions Justification

We request the minimum permissions required for functionality:

  • activeTab: Used to access the text of the exercise only when you explicitly interact with the extension.
  • storage: Used to save your extension settings (e.g., "Auto Skip" preference) locally on your device.

4. Data Security

  • All data is transmitted via secure HTTPS encryption.
  • Your personal API keys are not exposed; the extension uses a secure backend token exchange system.

1. Veri Toplama ve Kullanım Politikası

Neleri TOPLAMIYORUZ?

  • Tarama geçmişinizi veya ziyaret ettiğiniz URL'leri TOPLAMIYORUZ, saklamıyoruz veya iletmiyoruz.
  • Pearson/MyEnglishLab kimlik bilgilerinizi (kullanıcı adı/şifre) TOPLAMIYORUZ veya saklamıyoruz.
  • Çözdüğünüz soruların kayıtlarını sunucularımızda SAKLAMIYORUZ.

Neleri Topluyoruz?

  • Uzantı Kimlik Bilgileri: Yalnızca kimlik doğrulama ve kredi yönetimi için bu uzantı (myenglishlab.blog aracılığıyla) için oluşturduğunuz özel kullanıcı adını/şifreyi işleriz.
  • Geçici Sayfa İçeriği: "Çöz"e tıkladığınızda, mevcut alıştırmadaki metin bir cevap oluşturmak için geçici olarak işlenir. Bu veri, çözüm sağlandıktan sonra saklanmaz.

Yapay zeka destekli çözümler sunmak için bu uzantı, verileri aşağıdaki servis sağlayıcıya iletir:

  • Google Gemini API (Üretken YZ): İngilizce alıştırmalarının metin içeriği, analiz için güvenli bir şekilde Google'ın API'sine gönderilir.

3. İzinlerin Gerekçesi

İşlevsellik için gereken minimum izinleri talep ediyoruz:

  • activeTab: Yalnızca uzantıyla açıkça etkileşime girdiğinizde alıştırma metnine erişmek için kullanılır.
  • storage (Depolama): Uzantı ayarlarınızı (ör. "Otomatik Atlama" tercihi) yerel olarak cihazınıza kaydetmek için kullanılır.

4. Veri Güvenliği

  • Tüm veriler güvenli HTTPS şifrelemesi ile iletilir.
  • Kişisel API anahtarlarınız ifşa edilmez; uzantı güvenli bir arka uç token değişim sistemi kullanır.
